## `hermes config [CONFIGNAME] [NEWCONFIGVALUE]`

Change or print configs

```
USAGE
  $ hermes config [CONFIGNAME] [NEWCONFIGVALUE]

OPTIONS
  -h, --help     show CLI help
  -v, --verbose  Show debug level logs

EXAMPLES
  hermes config
  hermes config hermes.url
  hermes config docker.username tiagonapoli
```
